Diagrama de Relaciones

El sistema utiliza las siguientes relaciones:

  • Producto - Categoría: Relación Muchos a Uno (Many-to-One). Un producto pertenece a una categoría, y una categoría puede tener muchos productos.
  • Producto - Etiqueta: Relación Muchos a Muchos (Many-to-Many). Un producto puede tener múltiples etiquetas, y una etiqueta puede estar en múltiples productos.

Endpoints de Categorías

GET /api/categorias
Obtiene todas las categorías existentes.
POST /api/categorias
Crea una nueva categoría.
Body:
{
    "nombre": "Electrónicos"
}
GET /api/categorias/{id}
Obtiene una categoría específica por su ID.
PUT /api/categorias/{id}
Actualiza una categoría existente.
Body:
{
    "nombre": "Electrónicos Actualizados"
}
DELETE /api/categorias/{id}
Elimina una categoría por su ID.

Endpoints de Productos

GET /api/productos
Obtiene todos los productos existentes.
POST /api/productos
Crea un nuevo producto.
Body:
{
    "nombre": "Laptop",
    "precio": 1499.99,
    "categoria": {
        "id": 1
    }
}
GET /api/productos/{id}
Obtiene un producto específico por su ID.
PUT /api/productos/{id}
Actualiza un producto existente.
Body:
{
    "nombre": "Laptop Actualizada",
    "precio": 1599.99,
    "categoria": {
        "id": 1
    }
}
DELETE /api/productos/{id}
Elimina un producto por su ID.
GET /api/productos/{id}/etiquetas
Obtiene todas las etiquetas asociadas a un producto específico.

Endpoints de Etiquetas

GET /api/etiquetas
Obtiene todas las etiquetas existentes.
POST /api/etiquetas
Crea una nueva etiqueta.
Body:
{
    "nombre": "Oferta"
}
GET /api/etiquetas/{id}
Obtiene una etiqueta específica por su ID.
PUT /api/etiquetas/{id}
Actualiza una etiqueta existente.
Body:
{
    "nombre": "Oferta Especial"
}
DELETE /api/etiquetas/{id}
Elimina una etiqueta por su ID.
GET /api/etiquetas/{id}/productos
Obtiene todos los productos asociados a una etiqueta específica.
POST /api/etiquetas/{etiquetaId}/productos/{productoId}
Asocia una etiqueta existente a un producto existente.
DELETE /api/etiquetas/{etiquetaId}/productos/{productoId}
Elimina la asociación entre una etiqueta y un producto.
